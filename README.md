# 📰 NewsAppp

A React-based news aggregation application using React and Bootstrap to provide users with a streamlined experience. The app fetches and displays news across diverse categories including business, sports, technology, entertainment, general news, science, and health from a News API.

---

## 🚀 Features

- 🔍 Browse top headlines from various categories like business, sports, and technology.
- 🖱️ Infinite scrolling for seamless article loading.
- ⚡ Fast and responsive interface with React.
- 📦 Uses [NewsAPI](https://newsapi.org/) for real-time news data.

---

## 🛠️ Tech Stack

- React
- Bootstrap
- NewsAPI
